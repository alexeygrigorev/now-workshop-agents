Workshop Outline

Day 1: Foundations of AI Agents. (9:00 – 16:00)
● LLMs & RAG : Learn Large Language Models and Retrieval-Augmented
Generation. Build conversational agents using the OpenAI SDK and create
data-processing pipelines.
● Accessing the LLMs with OpenAI API
● Introduction to Retrieval Augmented Generation (RAG)
● Agentic Flows + MCP: Add agentic behavior with function calling, using libraries like PydanticAI, and Agents SDK. Expose tools via MCP.
● Agentic RAG workflows
● Advanced concepts beyond traditional RAG
● Pydantic AI as the agents orchestrator framework

Day 2: Building a Custom Coding Agent: (9:00 – 16:00)
● Setting up a Django project template
● Implementing tools for file access and manipulation
● Designing and developing a coding agent
● Coordinating multiple agents
● Monitoring & Guardrails: Use Pydantic Logfire for observability and safety.

Prerequisites:
Python, Visual Studio Code (or another Python editor), Docker and the OpenAI SDK.
